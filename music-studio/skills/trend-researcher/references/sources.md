# Approved sources

Every benchmark claim made by `trend-researcher` must cite one of these source types. The Sourcing quality gate will strip un-sourced claims.

## Reddit

- `r/edmproduction` — 1.5M+ subscribers; production-focused.
- `r/synthesizers` — gear and patches; signals about what's underserved.
- `r/audioengineering` — mixing, mastering, plugins.
- `r/ableton` — DAW-specific pain points.
- `r/wearethemusicmakers` — broader producer discussion.
- `r/edmproductionMP4`, `r/sounddesign` — short-form sound clips.

**Citation format:** post title + link + upvote count + date.

## YouTube

- Music-tech build channels (Andrew Huang, Look Mum No Computer, Benn Jordan, Alex Ball, Sam Battle, Ricky Tinez).
- Browser-audio creators (Tero Parviainen, Cory Ginsberg).
- Plugin reviewers (Pro Audio Tech, Streaky.com).

**Citation format:** video title + URL + view count + upload date + duration.

## GitHub

- `tonejs/Tone.js` — issues + discussions surface real workflow gaps.
- `webmidijs/webmidi` — what's hard about Web MIDI.
- Browse `github.com/trending?since=monthly&l=javascript` filtered for music keywords.

**Citation format:** repo URL + stars + recent issue thread URL.

## Hacker News

- Search `https://hn.algolia.com/?q=audio` and `?q=music+tool`.
- Comments often contain sharp critiques of tooling.

**Citation format:** post title + URL + comment count.

## Producer forums (lower priority)

- KVR Audio forums.
- Gearspace.

**Citation format:** thread URL + reply count.

## What does NOT count as a source

- "I read somewhere" — no.
- "The community says" — no.
- A single tweet from a non-notable account — no (tweets from established music-tech voices are okay if attributed).
- AI-generated content sites — no, they're often fabricated.

## Sourcing discipline

When citing:

> "Polyrhythmic sequencers are an underserved tool" → INSUFFICIENT.

> "Polyrhythmic sequencer requests have appeared in 4 r/synthesizers
> threads in the last 6 months: [link 1, 220 upvotes], [link 2, 180
> upvotes], [link 3, 95 upvotes], [link 4, 60 upvotes]." → SUFFICIENT.

The Specificity quality gate enforces this.
