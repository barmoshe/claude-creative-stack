# Packaging — title and thumbnail formulas

These are starting points. Replace with what's working on **your** channel after the first 5 episodes ship.

## Title formulas

Each formula sells the musical or surprising result first. Keep titles ≤ 60 chars (mobile truncation).

1. **"I built X that does [surprising musical result]"**
   _e.g._ `I Built a Sequencer That Writes Impossible Drum Patterns`
2. **"[Result], using [unexpected tool/method]"**
   _e.g._ `Generated a Techno Loop From a Pencil Drawing`
3. **"What happens if you [weird premise]?"**
   _e.g._ `What Happens If You Sequence Drums on a Spirograph?`
4. **"[Number] [things] in [time]"**
   _e.g._ `5 Drum Machines I Built in One Weekend`
5. **"This shouldn't work but [it does / does it?]"**
   _e.g._ `This Browser Synth Shouldn't Sound This Fat`

### Title scoring

Score each candidate on:
- **Curiosity gap** — would a stranger ask "wait, how?"
- **Clarity** — does someone reading it know what they'll watch?
- **Honesty** — does the video deliver this?
- **Mobile-truncation safety** — does it still work cut to 50 chars?

## Thumbnail formulas

Each formula has one core idea. Anti-list applies to all: no crowded code, no tiny plugin controls, no generic AI-art aesthetics, ≤ 4 words of headline text.

1. **App + 2-word headline.**
   App screenshot dominant; 2–4 words explaining the surprise.
2. **Before / after split.**
   Left: boring/expected thing. Right: weird/cool result. Arrow.
3. **Single weird visual.**
   The thing the title promises, made visible. No headline if the visual screams.
4. **Hand-drawn arrow + circle.**
   Authentic > polished. Points at the wow detail in the screenshot.
5. **Reaction face + result.**
   If the creator is on-camera. The face sells the surprise.

### Thumbnail scoring (Hard Rule 3)

A thumbnail must be **readable on a 350 px-wide phone preview in one second** before it ships. If the headline isn't readable at that size, the thumbnail fails.

## Anti-patterns

- Promises in the thumbnail or title that the video doesn't deliver.
- Generic "AI" framing as the headline.
- Code screenshots as the dominant thumbnail visual.
- Plugin GUIs at default zoom (illegible at 350 px).
