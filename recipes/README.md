# Recipes

End-to-end workflow narratives. Each one shows how to compose **knowledge + prompts + skills + artifacts + MCP** against a concrete goal.

| Recipe | Outcome |
|---|---|
| [`game-jam.md`](game-jam.md) | Build a playable single-file artifact game in a weekend. |
| [`animated-landing.md`](animated-landing.md) | Design and ship an animated marketing landing page. |
| [`data-story.md`](data-story.md) | Turn a messy CSV into an explanatory scrollytelling dashboard. |
| [`design-system.md`](design-system.md) | Generate tokens, a component kit, and a documentation site. |
| [`agentic-asset-pipeline.md`](agentic-asset-pipeline.md) | One-shot: brief → generated tiles → palette → atlas → live Kaplay artifact → vision critique. **Flagship demo** for the full stack working as one agent. |

These are deliberately "narratives not checklists" — the intent is to show the workflow, not prescribe it. Adapt freely.
